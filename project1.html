Project Overview

This project was a comprehensive database management assignment for CGS 2545C at the University of Central Florida, focusing on practical SQL implementation and database design principles.

The semester-long project was developed incrementally, with weekly milestones that built upon each other to create a fully functional database system with proper relationships and data integrity.

Project Requirements

The cumulative project assignment required me to:

  1. Design and implement a relational database using proper normalization principles
  2. Create multiple related tables with appropriate data types and constraints
  3. Write and implement database triggers for data validation and automation
  4. Insert and update data using various SQL manipulation techniques
  5. Generate a comprehensive Entity Relationship diagram to visualize the database structure and relationships

Key Learning Outcomes

This project provided hands-on experience with real-world database development, from conceptual design through implementation, updates, and maintenance.

Skills Developed

  • SQL Programming: Advanced query writing, data manipulation, and database operations
  • Database Design: Entity-relationship modeling and schema development
  • Data Integrity: Implementing constraints, triggers, and validation rules
  • Problem Solving: Troubleshooting complex database issues and optimization
  • Documentation: Creating clear technical diagrams and project documentation
  • Project Management: Following structured development timelines and milestones

Technical Implementation

The project utilized MySQL for database implementation, focusing on creating efficient, scalable database structures that could handle real-world data management scenarios. The final deliverable included a complete database schema with sample data, working queries, and comprehensive documentation. This project dealt with an airline for the database information and scenario. However, the skills and knowledge learned here can easily be adapted to many other fields that utilize databases.
